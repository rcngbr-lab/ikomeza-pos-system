# Customer Onboarding Checklist

## Before Go-Live

- Configure company name, branch, TIN, currency, tax settings, and users.
- Import products, categories, selling prices, costs, and opening stock.
- Create payment methods and reconciliation rules.
- Create stores: Main Store, Kitchen Store, Bar Store as needed.
- Train cashier, waiter, store keeper, manager, and admin roles.
- Run one full test sale, refund, shift close, stock-in, stock damage, backup, and restore verification.

## Go-Live Day

- Take pre-go-live backup.
- Lock old manual balances.
- Open shifts.
- Monitor errors, unmatched payments, stock deductions, and receipt printing.

## After Go-Live

- Review daily sales, VAT report, payment reconciliation, stock movements, and backup logs for the first seven days.
