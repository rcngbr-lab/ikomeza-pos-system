# SLA Draft

## Support Windows

- Standard support: business hours.
- Critical production incidents: same day response.

## Severity

- Critical: system down, data loss risk, payment processing blocked.
- High: checkout, refund, shift close, or stock receiving blocked for one branch.
- Medium: report discrepancy, UI issue, non-critical workflow interruption.
- Low: training, configuration, cosmetic issue.

## Commitments

- Backups must run daily.
- Production changes must be backed up before deployment.
- Restore procedure must be tested before commercial pilot.
- Payment reconciliation issues must be reviewed daily.
