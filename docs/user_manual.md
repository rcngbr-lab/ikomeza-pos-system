# User Manual

## Daily Flow

1. Login with username and password.
2. Open a shift before selling.
3. Use POS to search or scan products.
4. Add items to cart.
5. Choose payment method.
6. For mobile money, bank, or card, enter a payment reference or transaction ID.
7. Complete sale and print receipt.
8. Close shift at end of duty.

## Data Visibility

- Cashiers and waiters see their own operational data in their branch.
- Managers see assigned branch operations.
- Admins can view all branches and system modules.

## Important Controls

- Non-cash payments stay unmatched until reconciled.
- Refunds and stock adjustments are audited.
- Audit logs cannot be edited or deleted through normal application users.
